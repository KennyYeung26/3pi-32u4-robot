# 3pi+ 32U4 Robot

An autonomous robot programmed in C to navigate a coordinate grid and detect 
targets using five line sensors.

## How It Works
The robot moves to specific locations in a coordinate grid using precise motor 
control built on top of existing driver functions. It uses five line sensors to 
detect white squares (bombs) on the grid.

## Features
- Autonomous navigation to specific coordinate locations
- Bomb detection using five-line sensors
- Modular code structure built upon existing C driver functions for 
  precise movement control

## Tools & Languages
- C
- Pololu 3pi+ 32U4 Robot
